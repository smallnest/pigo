Design Decisions

Decision theme 仅存储，不接线发现

Ambiguity: pigo 目前没有 theme 运行时消费者。

Choice: 把 theme 包整树存到 $PIGO_HOME/themes/<name>/，不写启动器、不做发现接线。

Rationale: Issue/PRD 明确 theme 本期"仅存储待未来消费"。给它一个确定的家，让 install/list/uninstall/update 生命周期与其它类型保持一致。

Deviation None

Tradeoffs

Tradeoff 与 skill/theme 分发共用 copyTree

Alternatives: (A) 复用 copyTree（与 skill/extension 一致的整树复制 + 重装清理）；(B) 为 theme 单独写复制逻辑。

Winner: A。theme 与 skill 的分发形态一样（一个包目录 → 一个 <name>/ 子目录），共用同一原语。
