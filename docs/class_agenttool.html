9 个内置工具实现 agentcore.AgentTool，经 ToolRegistry 注册/校验，由 ToolExecutor 三阶段执行

类图 (Class Diagram) — 结构性图形。展现工具层静态结构：接口实现、注册表聚合、执行配置与 Todo 数据簇。

注册与校验

Registry · JSON Schema

  • 9 个内置工具实现 agentcore.AgentTool，各自声明 name / schema / 执行模式。
  • ToolRegistry 以 map[string]AgentTool 聚合工具，并预编译 jsonschema v6 校验器。
  • Validate 返回 []FieldError，经 ValidationErrorResult 转为错误结果。

三阶段执行与批处理

Executor · Batch

  • executeToolCall：prepare → execute → finalize，任何失败都编码为 IsError 结果而非 panic。
  • ToolExecutorConfig 携带注册表与三个可选 hook；BatchConfig 内嵌它并加 ForceSequential。
  • ExecuteToolCalls 按模式串/并行；含 sequential 工具则整批串行。
